Install python 3.8 from the zip file


Create a new folder in C:\ called "Crab Bot" (Directory will look like "C:\Crab Bot\[files]"

Open IDLE once Python installs

From IDLE, open C:\Crab Bot\bot.py

In Idle, choose "Run" from the options at the top, and choose "Run Module"

If successful, it should sit for a few moments and say "Logged on as Crab Bot"
